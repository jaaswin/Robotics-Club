# Embedded Systems Workshop

## Rajiv Gandhi College of Engineering and Technology

---

## Workshop Overview

The **Embedded Systems Workshop** was conducted at **Rajiv Gandhi College of Engineering and Technology** with the objective of providing hands-on exposure to embedded system fundamentals using **MicroPython** and the **Raspberry Pi Pico**. The workshop followed a structured four-day plan, progressing from basic concepts to complete mini-project implementation.

The sessions emphasized practical learning through simulations, hardware interfacing, and project-based activities.

---

## Platform and Tools

* Raspberry Pi Pico (RP2040)
* MicroPython
* Thonny IDE
* Wokwi Online Simulator
* LEDs, buzzers, buttons, joysticks, sensors, and display modules

---

## Day-wise Workshop Plan

### Day 1 – Foundations of Embedded Systems

* Introduction to embedded systems and their applications
* Microcontroller and microprocessor

  * Basic concepts and key differences
* Raspberry Pi Pico

  * RP2040 overview
  * Hardware features
  * Pinout and specifications
  * Applications
* Wokwi simulation

  * Introduction to online simulation
  * Simulating Raspberry Pi Pico projects
* Understanding GPIO pins

  * GPIO input and output
  * Pin mapping and pin numbering
* Installing MicroPython

  * Firmware installation
  * Setting up Thonny IDE
* First program

  * LED blink using MicroPython

**Outcome:**
Participants understood embedded system fundamentals and successfully executed their first MicroPython program on Raspberry Pi Pico.

---

### Day 2 – Digital and Analog Interfacing

* Digital input concepts

  * Push buttons
  * Switches
* Digital output control

  * LED patterns
  * Buzzer control
* Mini task

  * Button-controlled LED and buzzer
* Analog input and output devices
* Reading analog values

  * Joystick
  * Potentiometer
* PWM (Pulse Width Modulation) concept
* LED dimming

  * Using potentiometer and PWM

**Outcome:**
Participants learned to interface digital and analog devices and gained a clear understanding of PWM-based control.

---

### Day 3 – Displays, Sensors, and Interfacing

* Joystick interfacing with RGB LEDs
* Seven segment display

  * Working principle
  * Interfacing with Raspberry Pi Pico
* LCD display

  * Basic commands
  * Displaying text and values
* Sensors and communication

  * Overview of commonly used sensors
  * Sensor interfacing concepts
* Sensor-based mini projects

  * Distance measurement
  * Light or temperature monitoring

**Outcome:**
Participants gained hands-on experience with displays, sensors, and real-time data visualization.

---

### Day 4 – Mini Project Development

* Applying embedded system knowledge to real-world problems
* Overview of embedded project design
* Project planning

  * Problem statement
  * Block diagram
  * Flowchart
* Mini project implementation

  * Sensor-based or display-based projects
* Testing and debugging
* Optimization techniques
* Project demonstration
* Feedback and discussion

**Outcome:**
Participants completed functional embedded system mini projects and gained confidence in system design and implementation.

---

## Conclusion

The Embedded Systems Workshop was successfully conducted as planned and achieved its learning objectives. Participants developed a strong foundation in embedded systems, gained hands-on experience with Raspberry Pi Pico and MicroPython, and demonstrated their understanding through mini projects. The workshop encouraged practical thinking and motivated students to explore further applications in robotics, IoT, and automation.
